Use hand signals to control the presentation instead of your mouse!

Install these dependencies
-opencv-python
-cvzone

This accesses presentation files via folder assumed as "Presentation" 
Number your slides numerically 
Ensure Webcam is working 

Controls:
Go Back - Thumb
Go Forward - Pinky
Draw - Pointer Finger
Pointer - Pointer + Middle Finger
Erase - Pointer + Middle + Ring Finger
